# 鲜卉商城系统网上花店

## 项目概述

本项目是基于 Django 框架开发的 Python 鲜卉商城系统网上花店。该系统旨在为用户提供便捷的在线购花平台，实现花卉商品的在线展示、购买和管理等功能。

## 技术栈

- **后端框架**：Django
- **编程语言**：Python 3.7/3.8
- **数据库**：MySQL 5.7 或更高版本
- **数据库工具**：Navicat 11（推荐）
- **开发环境**：PyCharm、VS Code
- **前端框架**：Vue.js

## 功能模块

- 花卉商品展示
- 用户注册与登录
- 购物车管理
- 订单处理
- 用户评论

## 系统角色

- 管理员：负责商品管理、订单管理、用户管理等
- 用户：浏览商品、添加购物车、下单、评论等

## 运行环境

- Python 3.7 或 3.8
- MySQL 5.7 或更高版本
- Node.js（前端 Vue.js 需要的运行环境）

## 部署步骤

1. 准备 Python 和 MySQL 环境并进行配置
2. 克隆项目源码到本地
3. 安装依赖包
4. 数据库迁移与数据填充
5. 前端资源编译
6. 启动后端服务
7. 访问前端页面

## 目录结构

```
project/
│
├── backend/          # 后端代码目录
├── frontend/         # 前端代码目录
└── scripts/          # 项目脚本目录
```

## 核心亮点

- 基于 Django 强大的后端框架，保证系统稳定性
- 灵活的数据库设计，便于后期扩展和维护
